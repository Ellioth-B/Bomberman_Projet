BomberWild

How to play

Player 1

  • Arrow keys … move
  • Spacebar / alt … plant the bomb

Player 2

  • W, A, S, D … move
  • Shift … plant the bomb

Extra

  • M … mute / unmute sound
  • Enter … stop / restart
  • Escape … menu
